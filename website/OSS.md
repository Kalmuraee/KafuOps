# Open Source

KafuOps is intended to be open source.

## Why open source

Backend observability and debugging sit close to sensitive production systems. Teams should be able to inspect how the agent works, run it locally, self-host it, and control what data is sent to models.

## Community goals

- Transparent architecture.
- Extensible integrations.
- Public policies and safety mechanisms.
- Community scanners for different frameworks.
- Shared incident-memory patterns.

## Contributor areas

- Language scanners.
- Observability integrations.
- Git provider integrations.
- Redaction rules.
- Sandbox runners.
- Documentation.
- UI/dashboard.
- Policy engine.
