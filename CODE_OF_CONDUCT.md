# Code of Conduct

KafuOps should be a welcoming project for backend engineers, SREs, platform engineers, researchers, and open-source contributors.

## Expected behavior

- Be respectful.
- Assume good intent.
- Give constructive feedback.
- Keep discussions technical and inclusive.
- Respect privacy and security concerns.

## Unacceptable behavior

- Harassment.
- Personal attacks.
- Sharing private information.
- Encouraging unsafe production practices.
- Submitting intentionally harmful code.

## Reporting

Project maintainers should define a private reporting channel when the project is launched.
